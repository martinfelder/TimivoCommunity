# Timivo Community

Timivo ist eine native iOS-App zur Arbeitszeiterfassung, Auswertung und Verwaltung von Arbeitstagen, Abwesenheiten, Ferien und Überzeit.

Dieses Repository ist das öffentliche Community- und Roadmap-Repository für Timivo. Der App-Quellcode liegt bewusst nicht hier.

## Wofür dieses Repository gedacht ist

- Roadmap und geplante Funktionen
- TestFlight-Hinweise für Testerinnen und Tester
- Changelog und wichtige Neuerungen
- Feedback, Fehlerberichte und Funktionswünsche über GitHub Issues

## Aktueller Fokus

Timivo wurde grundlegend überarbeitet. Im Mittelpunkt stehen aktuell:

- klarere Navigation
- verbesserte Arbeitsübersicht
- Prüfansicht für auffällige oder unvollständige Einträge
- erweiterter PDF-Export
- bessere Analyse von Arbeitszeit, Abwesenheiten, Ferien und Überzeit
- Stabilität, Lokalisierung und TestFlight-Vorbereitung
